Themes

Hyper comes with multiple overall color schemes. Currently there are three themes available: Purple (Default), Dark and Green. You can switch to any of them by simply making one line change in the code.

Open src/App.js file and on top of the file, simply import the respective scss file. e.g. search for "DefaultTheme.scss". Here you can simply uncomment import statement for dark or green. Make sure to comment the import statement of other themes, otherwise your bundle would have css compiled for all the themes.

Default (Purple)

import './DefaultTheme.scss';

Dark

import './DarkTheme.scss';

Green

import './GreenTheme.scss';
